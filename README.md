 KENYAN_POPULATION
 The project demonstrates how to:

- Work with JSON data stored in a public GitHub repository.
- Use the Python requests library to fetch JSON data.
- Convert JSON responses into Pandas DataFrames.
- Save the cleaned data as CSV files.

Repository Contents

 File  Description 

population_data.json- Synthetic dataset created with Mockaroo and uploaded to GitHub
Python script that reads the JSON file from GitHub, converts it to a DataFrame, and saves as CSV.
dummyjson_products.py Script that fetches data from the DummyJSON Products endpoint and saves as CSV
dummyjson_carts.py  Script that fetches data from the DummyJSON Carts endpoint and saves as CSV
products.csv Output CSV from the Products endpoint
carts.csv`  Output CSV from the Carts endpoint
github_data.csv  Output CSV from the synthetic GitHub JSON data
